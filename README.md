## Setup Instructions

To run locally:

```bash
git clone https://github.com/SamiNachwati/samidev-portfolio.git
cd samidev-portfolio
npm install
npm run dev
Visit http://localhost:5173 in your browser.
```

## Deployment
This site is deployed via GitHub Pages using the ```bash gh-pages``` branch.
To redeploy:
```bash
npm run build
npm run deploy
```
